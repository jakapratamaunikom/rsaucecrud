## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rsaucecrud', git: 'git://github.com/nitzaalfinas/rsaucecrud.git'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rsaucecrud
```
